컴파일러는 기본적으로 정수 리터럴을 int형으로 인식함
long balance = 11111111111L; : long타입임을 명시하기 위해 정수리터럴 뒤에 L(l)표기해줘야함(롱타입 선언할때는 무조건 L써라)

실수는 일반적으로 double로 선언, 소수점이하 15자리로 허용범위가 가장크므로
float pi = 3.14F; : 실수리터럴은 기본적으로 double형으로 인식함, 표현하려면 F표기해줘야함

/n : 줄바꿈
/r : 홈키역할, 커서 맨앞으로
/b : 백스페이스, 지우기

+ 연산자 : 두가지 기능이 있음(오버로딩), 덧셈(숫자일때), 결합(글자일때)

연산자는 피연산자가 동일한 타입일 때만 연산함

형 변환 : 숫자 10이 내부적으로 글자로 연산됨 

논리곱 && : 모두 true인경우, 그리고 라는 뜻
논리합 || : 둘중하나 true인경우 or값

Short-circuit = 짧게수행되는 논리연산자인 &&, ||
