break : 대괄호 바깥으로 이동시킴 (continue와 반대개념)

continue : 대괄호 안쪽으로 이동시킴 (break와 반대개념), 건너뛰는 점프기능(skip)

label : 대문자로 쓰는게 관례

return; : 메소드 영역 바깥으로 이동시킴 = 메소드를 끝내는것

return 반환값; : 호출한 메소드에 값을 반환하며 끝남

매개면수 목록 : 외부에서 전달받는 목록

메소드 안에 넣는것 : 하나의 기능으로 만들어버림

메소드이름 : 함수이름, 식별자, 관련 이름으로 짓기

<객체지향의 목적>
- 변수 : 데이터 재사용
- 메소드(함수) : 기능의 재사용
- 객체지향 : 데이터+기능을 합쳐서 재사용, 재사용 할 수 있는 단위가 커짐

객체지향 4대특징
1. 추상화 : 일상생활의 객체를 class의 틀을 만드는 것
2. 캡슐화 : class안에 데이터와 기능이 하나의 캡슐화가 되어있는 것
3. 상속
4. 다형성

Dog.java : 객체 추상화 클래스
DogExample : 프로그램의 실행을 위해 만든것, 애플리케이션 클래스

레퍼런스 변수 : 인스턴스 자체가 할당되는것이 아니라 주소번지가 할당되는것임 ex. 바탕화면 아이콘
. : 객체를 호출할 수 있는 구분점
